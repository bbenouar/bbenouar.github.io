The app has since been enhanced with additional features such as goal setting, weight charting, and visual progress statistics, leveraging the power of cloud storage to improve data accessibility and scalability.

Enhanced Version

The Weight Tracker System was selected for my ePortfolio because it demonstrates key skills in mobile app development, database integration, and user interface design. The app uses both a local SQLite database for offline storage and Firebase database hosted by Firestore for cloud data persistence. Key features include user authentication, real-time chart updates, and saving goal weights. I enhanced the app by adding fields and buttons for seamless updates between the app and Firebase, improving real-time data sync across devices. This showcases my proficiency in cloud technologies and efficient data management.
