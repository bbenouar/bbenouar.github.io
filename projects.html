Using the same Weight Tracker System App, the project transitioned from initially utilizing a local SQLite database, which simply logged the entered weights, to a more organized cloud-based database hosted by FireStore. The app has since been enhanced with additional features such as goal setting, weight charting, and visual progress statistics, leveraging the power of cloud storage to improve data accessibility and scalability.

The Weight Tracker System was selected for my ePortfolio because it demonstrates key skills in mobile app development, database integration, and user interface design. Key features include user authentication, real-time chart updates, and saving goal weights.
